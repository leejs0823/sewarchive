# 📚 sewarchive (세와카이브)

> **보안(Security)과 웹(Web)을 엮어(sew) 지식을 아카이브(archive)하다.**  
> 개발, 디자인, 해킹을 함께 읽고, 기록하고, 성장하는 스터디
---

## 📖 스터디 소개

**세와카이브**는 **보안/웹** 트랙 모두가 참여하는 책 읽기 스터디입니다🤗

개발, 디자인, 해킹 관련 책을 함께 읽고  
**함께 리뷰**하는 것을 목표로 합니다.

---

## 🎯 이렇게 성장해요.

- 보안/웹 트랙 전문성 강화
- 개발, 디자인, 해킹 공부하기
- 책에서 얻은 인사이트를 나누기
- 꾸준한 기록을 통해 팜에서 성장하기!

---

## 🛠️ 간략한 활동 FLOW

- 자율 참여 방식 (고정 활동 시간 없음!)
- 테스트베드에서 책 대여 후 '책 대여 명부' 본인이 직접 작성 (리드미 맨 아래에 있어요!💚)
- 깃허브 레포지토리(세와카이브)의 책 이름 폴더 안에 개인 .md 파일을 만들어 리뷰 작성
- 리뷰 작성까지 마쳤다면, **파밍로그**에도 공유하면 good!

### 2. 개인 폴더 생성 및 리뷰 작성

- 레포지토리에 책 이름 폴더가 없다면 새로 생성합니다.
  예시: `/이펙티브자바`
- 폴더 안에 Markdown 파일을 만듭니다. '자신의 이름 + 파일(`.md`)'을 생성하여 리뷰를 작성합니다.  
  예시: `/이펙티브자바/김파머.md`
- 커밋메시지 작성 형식: ⭐️ [책 리뷰] 이름 - 책 제목

### 3. 리뷰 작성 가이드

- 읽은 책 제목 및 범위를 적어주세요.
- 핵심 요약 및 인상 깊은 내용을 작성해주세요.
- 개인적으로 얻은 인사이트 중심으로 기록하세요.
- 느낀 점 및 본인의 프로젝트에 적용 계획을 알려주세요.
- 자신의 블로그 링크가 있다면 첨부해주세요.
